# Dramabox Streaming

This is a Vue.js project for streaming dramas, using Bootstrap for UI.

## Project Structure

The project structure follows the `belajar-vuejs-contact-management` project.

- `src/components`: Contains UI components.
- `src/lib/api`: Contains API client code.
- `src/views`: (Not used, components are used directly in router).

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

## Features

-   **Home**: Displays Trending and Latest dramas.
-   **Search**: Search for dramas.
-   **Stream**: Watch dramas (supports episode selection).

## API

The app uses the API at `https://dramabox.sansekai.my.id/api`.
